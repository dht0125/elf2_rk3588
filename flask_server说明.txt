flask_server.py 说明
====================

flask_server.py 是本项目的后端服务主程序，基于 Flask 框架实现，主要功能包括：

1. 提供视频流管理与转发（支持 RTMP 拉流、本地摄像头等多种视频源）。
2. 通过 API 实现设备信息的读取与设置（集成阿里云 OpenAPI）。
3. 提供车辆检测数据的获取、统计与重置接口。
4. 支持前端页面展示视频流和检测结果。
5. 通过多线程和共享内存机制，实现高效数据交互。
6. 提供对外接口，便于前端或其他服务调用。

通过该 py 文件可以操控 ./board/MQTT/demos/data_model_basic_demo.c 等相关文件，实现数据模型的启动、停止、状态查询等操作。
